# 8-puzzle AI Solver

A solver for the 8-puzzle problem based on Artificial Intelligence.

# Usage

```shell
./TP1 <algoritm> <puzzle-entries> [PRINT] [-log-level=<log-level>]
                                  [PRINT_STATISTICS]
```

# Example

```shell
./TP1 B 1 2 3 4 5 6 7 0 8 PRINT -log-level=INFO
```
